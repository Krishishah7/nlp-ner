# Named Entity Recognition (NER)

This project demonstrates Named Entity Recognition in NLP using a pretrained spaCy model.

## Features
- Entity extraction from text
- Identification of people, organizations, locations, and dates
- Clean and interpretable output

## Tools Used
- Python
- spaCy
- Google Colab

## Purpose
NER is widely used in information extraction, search engines, resume parsing, and knowledge graph construction.
